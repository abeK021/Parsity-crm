* [Heroku](https://devcenter.heroku.com/)

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/connordipietro/parsity-crm.git
   ```
2. Install NPM packages for Client
   ```sh
   cd client
   npm install
   cd -
   ```
3. Install NPM packages for Server
   ```sh
   cd server
   npm install
   ```
